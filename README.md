# Postreria
Este sistema se realizo pensando en la solucion de ventas de un pequeño negocio de postres
El sistema cuenta con un login, el administrador puede agregar nuevos usuarios, y eliminarlos.
-Contien POS de veta
-CRUD de usuarios
-CRUD de productos
-CRUD de categorias
-Modulo de reportes: ganancias, numero de peditos realizados, promedio de ticket, producto mas vendido.
